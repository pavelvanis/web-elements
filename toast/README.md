
# Toast
This is library for toast

### Default toast settings
```js
const DEFAULT_OPTIONS = {
    autoClose: 5000,
    position: 'top-right',
    canClose: true,
    onClose: () => { }
}
```

> #### How create toast
> ```js
    new Toast({
        text: "Toast 1",
        position: "top-right",
        autoClose: 3000,
        canClose: false
    })```
>
